Learning Resources

Want to learn more about technology? Check out the following free resources:

  • Khan Academy

    Khan Academy offers free courses taught using a 'youtube video' style on subjects including math, science, programming, test prep for college entrance exams, economics, as well as the arts and humanities.

  • Code.org

    Code.org provides a fun, interactive way for school-aged students to learn to code. Their 'Hour of Code' library includes hour-long tutorials designed for all experience levels giving hands-on practice with languages like Blocks, Scratch, JavaScript, Python, and Java.

  • Codecademy

    Codeacademy also offers interactive courses that covers languages that include HTML, CSS, JavaScript, Ruby, Python and many more. These courses provide an excellent way for students to familiarize with themselves with programming fundamentals and the syntax of specific programming languages.

  • Coursera

    Coursera offers a multitude of free online courses taught by prestigious universities in subjects such as Computer Science, Data Science, Finance, and more.
